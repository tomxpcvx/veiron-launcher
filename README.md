# veiron-launcher 

Launcher for the veiron game.

**This source code is already some years old and absolutely badly programmed. (one of my first java projects)**

## Useful Links
* **Downloads** - <https://github.com/tomxpcvx/veiron-launcher/releases>

## Build
veiron-launcher uses Maven as build system.

### Requirements
* Git
* Java 11
* Maven 3.6.0 (used for development, other will work too)

### Checkout sources
Open a shell and type:
```
git clone https://github.com/tomxpcvx/veiron-launcher.git
cd veiron-launcher/
mvn clean package
```
You can find the jar in the `target/distribution` directory.
